Instruções para reviewer da Meta

Objetivo: Este app demonstra o uso da feature “Page Public Content Access”. A funcionalidade principal do app é buscar, exibir e analisar posts públicos de páginas.

Passos para teste:

  1. Clique em “Entrar no painel”.
  2. Na tela principal, insira um Access Token válido no campo correspondente.
  3. Informe o ID ou username de uma página pública do Facebook.
  4. Clique em “Buscar posts públicos”.
  5. Verifique a listagem de posts públicos retornados.
  6. Verifique a seção “Análise”, que resume a atividade pública da página.
  7. Caso a API não retorne dados no ambiente de demonstração, visualize a seção de resultados demonstrativos com dados fictícios claramente rotulados como “DEMO”.

Como este app usa os dados

Atenção: Este aplicativo usa dados públicos de páginas do Facebook para analisar e/ou exibir posts e engajamento públicos.

O objetivo principal é agregar conteúdo público para monitoramento, extraindo texto do post, data de criação e métricas agregadas (reações, comentários e compartilhamentos), provendo insights analíticos sobre a página.

Dados Utilizados da API

  • id da Página/Post
  • message (Texto do post)
  • created_time (Data de publicação)
  • permalink_url (Link do post)
  • Métricas de Engajamento: reactions, comments, shares
